# UNIKEY

A library for unique code generation

## Install

For npm:

```bash
npm i unikey
```

For yarn:

```bash
yarn add unikey
```

For pnpm:

```bash
pnpm add unikey
```

## Usage

```js
import unikey from 'unikey';

const key = unikey(); // 2YsCOcON

const key16 = unikey(16); // hFkcpCtU3AWILeQI

// ...
```
